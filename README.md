The Secure Python CLI Password Manager is a simple Python command-line program.
Its main purpose is to offer a safe local solution for storing and retrieving private login information.
It places a great emphasis on security by using robust encryption for all data stored on the local disk,
ensuring that the passwords are safe even if the storage file is viewed.

Features used in this project are as follows:

-Symmetric Encryption
-Secure Input
-Persistent Storage
-Key Management
-CRUD Operations

Steps to Install & Run the Project
Prerequisites
Python 3.x installed on your system.

Access to a terminal or command prompt.

-Save the Code
Save the main code logic into a file named password_manager.py.

-Install Dependencies
Open your terminal or command prompt, navigate to the project directory, and install the required external library

-Execute the Program
